# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-20

### Added
- Initial release of RepoLens
- Repository scanning with file and folder analysis
- Language detection for 50+ programming languages
- Git statistics (commits, branches, contributors)
- Dependency analysis (unused, missing, outdated)
- Security scanning (hardcoded secrets, tokens, keys)
- Code quality metrics (circular imports, complexity, duplication)
- TODO/FIXME comment detection
- Documentation audit (README, LICENSE, .gitignore, tests, CI)
- Multiple output formats: Terminal, JSON, Markdown, HTML
- Project health scoring (0-100) across 8 categories
- Interactive CLI with progress spinners and colored output
- HTML report with dark/light mode and responsive design
